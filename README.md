# Site-de-p-gina-presente-de-aniversario
Este projeto é um modelo de página-presente de aniversário desenvolvido com HTML e CSS. O objetivo é oferecer uma forma simples e personalizada de homenagear alguém especial por meio de uma página web com mensagens, fotos e elementos visuais comemorativos.

Funcionalidades
Layout responsivo para diferentes dispositivos.
Exibição de fotos especiais.
Mensagens personalizadas de aniversário.
Design simples.
Fácil personalização de textos, imagens e cores.

Tecnologias Utilizadas
HTML5
CSS3

Objetivo do Projeto
Este projeto foi criado como modelo para páginas-presente de aniversário, permitindo que qualquer pessoa adapte o conteúdo para celebrar momentos especiais de forma criativa e única.
